Content 영역(innerHTML, innerText) 관련 메서드

* html("문구") 메서드 : innerHTML 속성과 관련된 메서드

선택된 요소의 content 영역(시작과 종료 태그 사이)을 리턴해 주거나 또는 변경해 주는 메서드
- $("선택자").html() : 선택된 요소의 content 영역의 값을 반환
- $("선택자").html("문구") : 선택된 요소의 content 영역에 값을 문구로 변경해 줌
전달하는 문구에 html 태그가 있을 경우 태그로써 해석이 된다.

* text("문구") 메서드 : innerText 속성과 관련된 메서드

선택된 요소의 content 영역의 텍스트만 리턴해 주거나 변경해 주는 메서드
- $("선택자").text() : 선택된 요소의 content 영역을 반환(단, html 태그는 무시되고 텍스트로만 전달됨)
- $("선택자").text("문구") : 선택된 요소의 content 영역을 전달값으로 변경
