* 삽입 관련 메서드 part 1

선택된 요소를 기준으로 새로이 생성된 요소를 추가하는 메서드

$(A).append(B) : A 요소 내부에 가장 뒤에 B를 추가(자손/하위)
$(A).pripend(B) : A 요소 내부에 가장 앞에 B를 추가(자손/하위)
$(A).before(B) : A 요소의 앞에 B를 추가(동위)
$(A).after(B) : A 요소의 뒤에 B를 추가(동위)

* 삽입 관련 메서드 part 2

$(B).appendTo(A) : B를 A요소 내에 뒷 부분에 추가(자손/하위)
$(B).prependTo(A) : B를 A요소 내에 앞 부분에 추가(자손/하위)
$(B).insertBefore(A) : B를 A요소 앞에 추가(동위)
$(B).insertAfter(A) : B를 A요소 뒤에 추가(동위)

let 변수 = $("선택자").colone(true / false) : 선택된 요소를 복제해서 반환해 주는 메서드
=> true / false : 해당 선택된 요소에 걸려있는 이벤트까지 복사할 건지 지정(기본값 : flase, 생략 가능)

* 요소 객체 제거 및 잘라내기 메서드

$("선택자").empty() : 선택된 요소의 모든 하위 요소들을 제거시켜 주는 메서드
let 변수 = $("#선택자").remove() / .detach() : 해당 선택된 요소를 제거 후 제거된 요소를 반환
- .remove() : 잘라내기 시 잘라낸 요소의 이벤트를 안 가져옴
- .detach() : 잘라내기 시 잘라낸 요소의 이벤트까지도 가져옴
