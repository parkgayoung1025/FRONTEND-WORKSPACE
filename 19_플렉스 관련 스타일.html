display : flex

        선택자 {
            display : flex;
        }
        * 요소를 통해 레이아웃을 구성 시 유연하게(flex) 위치를 선정할 수 있도록 도와주는 속성
        단, flex 요소끼리만 영향을 끼침

justify-content : 요소들을 가로로 정렬하도록 도와주는 속성

        선택자 {
            justify-content : flex-start/flex-end/center/space-between/space-around/space-evenly
            
            flex-start : 요소들을 왼쪽으로 정렬
            flex-end : 요소들을 오른쪽으로 정렬
            center : 가운데 정렬
            space-between : 요소들 사이에 동일한 간격을 부여하여 정렬
            space-around : 요소들 주위(양 옆)에 동일한 간격을 부여하여 정렬
            space-evenly : 요소들마다 동일한 간격을 부여하여 정렬
        }
FLEX
FLEX

flex-direction : flex 요소들의 정렬 방향 지정(행, 열 단위)

        선택자 {
            flex-direction : row/row-reverse/column/column-reverse

            row : 요소들을 텍스트 방향과 동일하게 정렬
            row-reverse : 요소들을 텍스트의 반대 방향으로 정렬
            column : 요소들을 위에서 아래로 정렬
            column-reverse : 요소들을 아래에서 위로 정렬
        }

align-items : 요소들을 세로 기준으로 정렬하게 도와주는 속성

        선택자 {
            align-items : flex-start/flex-end/center/stretch

            flex-start : 요소들을 맨 위로 정렬
            flex-end : 요소들을 맨 아래로 정렬
            center : 요소들을 가운데로 정렬
            stretch : 요소들을 컨테이너(부모 요소)의 크기에 맞도록 늘려주는 속성
        }

flex-wrap : 요소들을 한 줄로 정렬할지, 여러 줄로 정렬할지 정하는 속성

        선택자 {
            flex-wrap : nowrap/wrap/wrap-reverse;

            nowrap : 요소들을 한 줄로만 정렬시키는 속성
            wrap : 요소들을 여러 줄로 정렬시키는 속성
            wrap-reverse : 요소들을 여러 줄로 반대로 정렬시키는 속성
        }
