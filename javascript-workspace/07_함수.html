함수(function)

* 선언적 함수

        function 함수명(매개 변수, 매개 변수, ...){
            해당 함수 호출 시 실행할 소스 코드;

            return 결괏값;
        }

        - 매개 변수가 없는 경우 매개 변수 생략 가능
        - 반환할 결괏값이 없는 경우 return 결괏값 생략 가능

* 익명 함수

        function(매개 변수, 매개 변수, ...){
            소스 코드;
        }

        특정 변수나 속성에 대입되는 함수 제시 시 주로 사용됨
        (주로 이벤트 핸들러 작성 시 사용)
