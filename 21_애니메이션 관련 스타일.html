애니메이션 관련 스타일

애니메이션

속성을 점차적으로 변화시켜 움직이는 효과를 내는 기법
트랜지션과 비교했을 때 시작 스타일과 종료 스타일을 부드럽게 이어주는 기능을 하는 것은 같지만
애니메이션 중간에 원하는 위치에서 @keyframes를 이용해서 중간 스타일 지정 가능

keyframes

애니메이션 시작 지점과 종료 지점의 스타일을 정하고 key frames의 이름을 지정
시작 지점과 종료 지점 두개만 설정할 경우 from to로 설정
중간에 작업이 더 있다면 %로 구분하여 설정 

        @keyframes 키프레임명 {
            선택자 {스타일;}
        }
